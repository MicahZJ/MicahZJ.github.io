# 前端旅程

# 2016，哒哒哒，入坑之始

从大三的时候接触到一门应该是叫网页设计与基础？的课程，开始接触html标签，一开始都是叫什么frame？那时候是16年吧，好像暴露年龄了。。。 再看看网上一些网站的效果，我怀疑我是上的一门假的课程。 那时候比较爱玩游戏，经常游览于各大网络游戏门户网站，是的，我可以毫不夸张的说，只要市面上有的游戏，我大部分都下载下来玩过。 就拿鹅厂的撸啊撸和天刀两个游戏来说，官网首页不要太酷炫，我上的课程怎么会这么low呢 然后我就开始了自学之路

那时候在慕课网啊，是的在这里感谢一波慕课网免费教程，虽然现在好像大部分收费了，那时候真的是看视频自己学啊，一遍又一遍的看教程熟悉html标签， 熟悉css语法，熟悉各种div+css布局方式，三栏两栏，上中下布局。可以说是从早到晚，除了上课，其实那时候大三课程也不多，所以大部分时间都是在看视频跟着敲代码 在经历一学期摧残之后，差不多知道css里面大部分常用的属性是用来做什么的了，最后网页设计与基础这么课，通过自己的努力写了一个静态网站出来，那页面现在看来简直爆炸 ，惨不忍睹，不忍直视的那种，挺搞笑的。大三一过，紧接着就开始准备实习了哇

实习期，可以说是真正进入前端这个大坑的开始。大四了学校不是要实习嘛，奈何实力不济，没有通过校招进入一家公司，虽然都是培训公司。。。怎么办呢，不能不实习吧 后来我小姑知道了这件事，说可以去她公司试试，也是一家不大不小的媒体公司，里面有前端，哦，那时候是网页设计的工作，可以推荐我去实习看看 这个嘛，就进了哇，然后就遇到了带我入门的老司机，亦师亦友的大圣，现在我人在上海，也是经常和他联系的，怎么说呢，进了公司之后，也没什么大活，就写写页面 真的是练习切图，会把他之前做的小项目给我练手，从简单的切图开始，每一个页面都得改三次左右，也谢谢他的严格，不然还真的一点技术水平都没有。

html+css布局练好了，练习什么呢，当然是js了。这也是一门神奇的语言，后话再题。怎么学呢，很简单，从一个轮播图开始，到一个轮播图结束，现在这个基本不用自己写了，但是那个时候 为了熟悉js，是自己一点一点手写的，其中心酸不足为外人道也。主要是真的不懂，只能一遍又一遍的去问，然后呢，也幸亏大圣一遍又一遍给我解释，哈哈。。想想我师傅他也 真的有够耐心的。

扯一下题外话，为什么叫他大圣，其实一开始我是叫他大神的，但是后来发现他姓孙，大神，大神喊着，喊着就成了大圣了，反正我是顺口了。 其实办公室里的人都不错，教我php的玉妹老师，那时候听说写了好多年的，反正是挺厉害的，还有运维吴叔叔哇，看上去挺和蔼可亲的一个人 ui黄庆老师也不错啊，虽然不是经常接触，不过人还是特别好的。同办公室的还有好几位老师。我是把所有人都定义为老师的，毕竟自己去实习， 可以从所有人学到东西

# 2017，毕业既失业，步入上海

转眼17年6月份就毕业了，这个时候就得找工作了，南京就先不考虑了，当时实习去投简历，一下子就面了两家培训公司，被套路一波。江苏省内的能搞IT的也就 无锡，苏州，南京，都去了一遍最后无功而返。最后和舍友聊了聊，决定一起去上海闯一闯。

刚毕业，手里钱也有限，投简历投了一个多礼拜，舍友先放弃了，经济是一部分的原因，这么算两个人住200多的快捷旅店，很便宜了把，再加上吃饭什么多，两个人各自也有花了1500软妹币的样子了 再加上确实没几家公司面试成功，大多简历都石沉大海了，于是舍友就回去了。

那我能回去嘛，不能啊，我这么，这么，这么，好吧，其实是因为我学都东西，如果要对口都化，必须去大城市，而上海是为数不多较近都选择，另外一个是杭州。所以 我打算在上海多待一段时间，但是钱不够，怎么玩呢？是都，我就住了一段时间青年旅社，是比较便宜都，60，70软妹币一晚，期间遇到了很多天南海北的人，有背包客，有已经工作的人，也有和我一样出来 找工作的人，形形色色的人都遇到了，其中心酸。。。其实没多大心酸吧，毕竟也不是娇生惯养长大得，一笔带过了。

在青旅又待了一个礼拜之后，终于找到了工作，虽然薪水不高，但是也够养活自己了。说实话对自己来说，也是抱着试一试的态度上班的，还是觉得自己的水平不行，可能会被辞退吧，毕竟是上班，要创造效益， 不然公司白养你啊。不过在自己战战栗栗过了三个月试用期后发现，自己应该是跟上公司的节奏了，也感谢一路带我的满大师和任大佬以及帅气吴的各种支持吧。因为，在项目初期，自己就是小白，肯定很多不懂 的问题，然后这些问题也比较基础，但是，他们还是愿意解答我，可以说公司的同事还是都很可爱的。说到可爱，还有一位四海兄弟，当初问他，你真叫四海吗，他回答是的。我又问他，你爸妈取这个寓意是什么？ 他说四海嘛，四海为家？五湖四海嘛，哈哈

当然，在这家公司也经常加班吧，都是些重复的劳动，一开始一位很难，其实上手了之后也就这么回事， 包括后面跟着外派回来的健哥，熊哥做项目，难也不难，但就是时间紧，慢工出细活的情况下，还要赶进度 就需要加班了，所以有一段时间还是经常加班的。

就这样该加班，该上班上班，正式工作5个月之后，我发现日子不能这么过。然后呢，就跟大圣讨论一下，他也觉得是这么回事，然后就丢过来一堆教程。白天工作内容就那些，然后我就开始学习新的前端知识，比如node，比如vue，比如自动化，比如npm包管理，比如git，比如小程序开发。整天用js操作dom的时代早就过去了 框架时代，我不能一直用旧的东西，前端日新月异，不说用最新的东西吧，总得用些主流的东西吧，然后又开始辛苦的学习之路。白天工作，晚上下班就埋头学习。真的觉得做IT不能不学习，不学习就废掉了。 时常关注一下互联网动态还是有帮助的，不然现在用的这个vuepress，怎么知道的，对吧

在学习中开始，在学习中进步，时间一晃就过年了，过完年我就辞职了，给自己放了一个假，春节过完，又补了一个长假。

# 2018，新的起点，新的开始

老实说，这一个月休息的我真心是要荒废了，差点不想上班，可是不上班又能干嘛呢，然后就赶紧找工作了，正好又是银四的找工作季节。 断断续续两个礼拜之后，在几份offer之中选择了现在的公司。

新的公司，新的起点，新的同事，希望自己能扎实的进步，未来是怎样的，没有人知道，所以，该努力努力，该学习学习，该休息休息。。。
